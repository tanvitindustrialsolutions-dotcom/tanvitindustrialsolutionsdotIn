Site par changes (admin)

Ye sirf rasta dikhane wali file hai. Asli admin andar chalta hai jab computer par chhota server ON ho.

Pehla kaam (sirf ek baar): project folder mein terminal kholo, npm install likho, Enter.

Roz ka kaam — do mein se koi ek:

  1. Windows: folder mein start-admin-local.bat par double-click karo. Ek window server chalayegi, browser mein admin khul jayega.
  2. Ya manually: terminal mein npm run server likho (server chalu rahe tab tak). Phir neeche wale button dabao.
Admin panel kholo (password se login)

Agar button kaam na kare to pehle server start karo. Port alag ho to URL mein 8787 badlo.

English: This page is only a shortcut. Saving products/clients/site data needs the small Node server running (npm run server or the .bat file). Then open /admin/ with your password — changes write into the data/ folder on this computer. For the live internet site, the same project must run on your host (see docs/ONLINE_ADMIN_DEPLOY.md).
