# Quotation Generator — SaaS Product

A professional quotation, invoice, proforma and purchase order generator built for Indian SMBs.

## Current Status
- ✅ Phase 1 Complete — Fully functional offline tool
- 🔄 Phase 2 — Planned (User auth + cloud storage)
- ⬜ Phase 3 — Planned (Razorpay billing + SaaS)

## Phase 1 Features
- Quotation, Proforma Invoice, Tax Invoice, Purchase Order
- Company profiles with logo and branding
- Client database with GSTIN validation
- Line items with drag-to-reorder
- Auto IGST / CGST+SGST tax detection
- Save history with Draft/Finalized status and remarks
- PDF export
- Fully offline — data stored locally via File System Access API

## Tech Stack (Phase 1)
- Pure HTML, CSS, JavaScript — single file, no dependencies
- File System Access API for local storage
