Same website as the public pages — URL path is /admin/ only (no separate domain). Edit products and valued clients here; use Site & clients for the home page (hero and text). The live server must run npm start (or npm run server) so saves go to the data/ folder on that machine.

Home page: hero, text, About & Contact

This is stored as one file on the server. Open the section below only when you need to change slides, wording, or contact details.

Advanced: edit site configuration as JSON

Up to 6 slides; each slide needs an image path like assets/slider/your-file.jpg. Put new hero images in the assets/slider folder on the server (same way you deploy the site), then type that path here. Optional pages.about and pages.contact override About and Contact. Links must be relative (no http://).

Valued clients (logos on the home page)

One entry per company. Click a row to expand and edit; only one stays open at a time. Short id is a slug (lowercase, numbers, hyphens). Use Upload logo inside an open row to save into assets/clients/, or paste a path if the file is already there.
